Wir treffen uns

Send an ics calendar file to a chat. Members can download the file and click it to add the appointment to their personal calendar.

Wird nur verwendet, wenn die Location auf online steht.
Hinweis: Der Raumname sollte so gewählt werden, dass er nicht erraten werden kann.
Bei einem Update die Versionsnummer erhöhen.
Der Button erzeugt einen Chat-Entwurf mit .ics-Datei.
