# meet up (webxdc)

Web form for creating a calendar event as an `.ics` file and sending it to a chat.

<img width="272" height="425" alt="Bildschirmfoto vom 2026-03-28 17-24-24" src="https://github.com/user-attachments/assets/09746cf2-bb48-4de7-91df-b17584a4718e" />

## Features

### Switchable languages: Deutsch (de), English (en), Nederlands (nl), Français (fr). Default: English.

### Event fields: title, description, date, time, duration, timezone, and version.
<img width="668" height="1065" alt="create_appointment" src="https://github.com/user-attachments/assets/a9272c39-5536-43c5-a50b-f90fdb4d6c2c" />


### Location type: `offline` or `online`.


#### For `online`: Jitsi server selection, optional custom server, room name, and generated meeting link.
<img width="667" height="240" alt="jitsi" src="https://github.com/user-attachments/assets/e3956fac-63e4-4f12-927c-371010196335" />

#### For `offline`: free text place input.
